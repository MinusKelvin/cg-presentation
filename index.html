How Computer Graphics Work
Start
Your screen is a grid of pixels, and so all computer graphics must be put of a grid of pixels somehow.

The screen can be viewed as a coordinate system.
Using whole-number coordinates, points fall exactly on pixels' centers and everything is really easy.

Drawing lines is simple, find the line's slope and step along it until you reach the other end.
